Обробник Подій

Обробник подій - пак, створений для роботи з подіями

# Приклад використання:

```mavka
;; оголошення обробника подій (можна вказати список подій, ак аргумент)
обр = ОбробникПодій()

;; додавання події (зазвичай повертає значення в зворотній виклик)
обр.додати_подію("привітання", дія() 
	вернути "привіт, Лесю"
кінець)

;; обробка події (можна приймати аргументи з обробника події)
обр.обробити("привітання", дія(привітання)
	друк("привітання отримано: %(привітання)")
кінець)
```

# Довідка
```mavka
структура ОбробникПодій
	події словник ;; тут зберігаються усі події
кінець

дія ОбробникПодій.викликати_створення(події словник) ;; оголошення обробника подій. так само, як і ОбробникПодій()

дія ОбробникПодій.додати_подію(назва текст, обробник Дія) ;; додає певну подію до обробника подій

дія ОбробникПодій.обробити(назва текст, зворот_виклик Дія) ;; оброблює певну подію за допомогою дії зворонтнього виклику
```
